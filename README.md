## Features
```
- auto detect installed browsers
- auto remove tracking elements from URLs (basic)
- !bangs
- ?search engines
- ...
```

## Requirements
```
- rofi : menu
- mpv  : video player
- sxiv : image viewer
```

## Installation
```
git clone https://github.com/MyOS-ArchLinux/nbrowser
cd nbrowser/
sudo make install
```
